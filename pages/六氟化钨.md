### 一、 化学性质 -
	- 六氟化钨（Tungsten Hexafluoride，化学式 WF₆）是一种非常特殊的化合物：
	  
	  | 性质 | 数值 / 描述 |
	  | ---- | ---- | ---- |
	  | **外观** | 常温常压下无色气体，低温加压为浅黄色液体，固态为白色结晶 |
	  | **分子量** | 297.84 |
	  | **熔点** | 2.3℃ |
	  | **沸点** | 17.5℃（夏天室温就能气化） |
	  | **气体密度** | 12.4 g/L（已知密度最大的过渡金属气体） |
	  | **分子结构** | 正八面体构型，W-F 键长 183.2 皮米 |
	  | **化学特性** | 强氟化剂、强腐蚀性、有毒，遇水极易水解，遇湿空气生成白色酸雾 |
	  | **CAS 号** | 7783-82-6 |
	- **反直觉的特点**：钨是熔点最高的金属（3422℃），但六氟化钨却是常温下的气体，沸点只有 17.5℃。这一特性恰恰是它能用于半导体制造的关键。
- ### 二、核心应用 -
	- 六氟化钨是**芯片制造中唯一成熟的金属钨沉积前驱体**，95% 以上用于半导体行业，是先进制程不可或缺的关键材料。
	- 核心用途：化学气相沉积（CVD）钨金属薄膜，**简单说就是给芯片内部 "填坑"**：在纳米级的微小孔洞 / 沟槽中沉积出高导电性的钨层，构建芯片内部的 "电路高速公路"。
	- 化学反应原理
	  ```
	  WF₆ + 3H₂ → W + 6HF
	  ```
		- 六氟化钨气体进入反应腔
		- 氢气把氟 "夺走"
		- 金属钨均匀沉积在晶圆表面的孔洞中
		- 副产物氟化氢（HF）被抽走排出
	- 为什么不可替代？
		- **台阶覆盖性极好**：能均匀填满深宽比极高的纳米级小孔
		- **电阻率低**：钨的电阻率约 5.6×10⁻⁸ Ω・m，导电性能优异
		- **热稳定性好**：耐高温，后续工艺中不会变形或扩散
		- **工艺成熟**：经过几十年量产验证，可靠性极高
- ### 三、纯度等级与技术壁垒
	- 六氟化钨的纯度直接决定了芯片良率，是技术壁垒最高的环节：
	  | 纯度等级 | 纯度标准 | 应用场景 |
	  | ---- | ---- | ---- |
	  | **5N 级** | 99.999% | 成熟制程（28nm 以上） |
	  | **6N 级** | 99.9999% | 先进逻辑、3D NAND、HBM |
	  | **7N 级** | 99.99999% | 3nm/2nm 最先进制程 |
	- #### 核心技术难点
		- **杂质控制**：金属杂质要控制在**ppb 级**（十亿分之一），一粒微尘就能废掉一整片晶圆
		- **提纯工艺**：需要多级精馏、吸附、过滤等复杂工艺链
		- **充装与运输**：超高纯气体对容器、管道、阀门都有特殊要求，避免二次污染
		- **分析检测**：要有能力检测出 ppb 级的杂质，检测设备本身就是壁垒
		  **技术代差**：国内多数企业仅能稳定量产 6N 级，中船特气是全球唯一能量产 7N 级的企业，技术领先同行 5-8 年。

	- #### 全球总产能

	  | **SK Specialty** | 2,000 | 韩国 | 主要自用供应 SK 海力士，外供极少 |

		- 原因：中国对钨原料实行出口管制，日本无钨资源、无替代渠道
		- 结果：全球供给直接出现约 30% 的刚性缺口，定价权加速向中国转移
- ### 六、产业链上下游
	- #### 上游：原材料
		- **钨粉 / 钨原料**：最核心原材料，中国占全球钨资源 80% 以上
		- **氟气 / 氢氟酸**：氟化反应原料
		- **电力**：生产过程高耗能，电价是重要成本因素
	- #### 中游：气体生产与提纯
		- 合成反应 → 粗馏 → 精馏提纯 → 充装检测
		- 技术壁垒最高的环节，也是利润最丰厚的环节
	- #### 下游：应用领域
	- **集成电路**（~70%）：逻辑芯片、存储芯片、HBM
	- **显示面板**（~15%）：LCD/OLED 面板制造
	- **光伏新能源**（~10%）：太阳能电池
	- **其他**（~5%）：科研、军工等
- ### 七、总结
  
  六氟化钨是**半导体先进制程的 "隐形基石"**，虽然用量不大（一片晶圆只用几克），但不可或缺。2026 年日本产能永久退出后，全球供需格局发生根本性逆转，中国企业从 "追赶者" 变成 "全球供给支柱"，定价权和利润空间大幅提升。
  
  **核心看点**：技术壁垒极高（7N 级全球独家）、供需缺口刚性（30%+）、国产替代加速、价格弹性巨大。
