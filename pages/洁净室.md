### 一、什么是洁净室

**洁净室（Cleanroom）**，也叫无尘车间、净化车间，是指将一定空间范围内的空气中的微粒子、有害空气、细菌等污染物排除，并将室内温度、洁净度、室内压力、气流速度与分布、噪音振动及照明、静电控制在某一需求范围内的特殊设计房间。

**一句话理解**：
**芯片制造的"无菌手术室"，空气中的灰尘少到极致，否则一颗灰尘就能毁掉一颗芯片。**

**为什么重要**：
	- 芯片的线宽只有几纳米（7nm、5nm、3nm）
	- 一颗0.1微米的灰尘，相对于芯片来说，就像一块大石头
	- 灰尘落在晶圆上，会导致短路、断路、良率下降
	- 先进制程的芯片厂，良率从0%提升到85%以上，洁净室是关键因素之一

- ### 二、洁净室的分级标准
	- #### 1. 国际标准：ISO 14644-1
	  
	  按**每立方米空气中≥0.1μm的粒子数量**划分，从ISO 1级到ISO 9级，数字越小越洁净。
	  | ISO等级 | 每立方米≥0.1μm粒子数上限 | 对应传统叫法 | 典型应用 |
	  |---------|--------------------------|-------------|---------|
	  | **ISO 1级** | 12颗 | 十级/一级 | 光刻核心区、掩膜版存放区、最先进制程 |
	  | **ISO 2级** | 120颗 | - | 部分先进制程关键区域 |
	  | **ISO 3级** | 1,000颗 | 百级 | 7nm及以下先进制程、光刻、刻蚀 |
	  | **ISO 4级** | 10,000颗 | 千级 | 28nm-14nm制程、晶圆制造大部分区域 |
	  | **ISO 5级** | 100,000颗 | 万级 | 成熟制程、封装测试、面板 |
	  | **ISO 6级** | 1,000,000颗 | 十万级 | 普通电子、医药 |
	  | **ISO 7级** | - | 百万级 | 食品、化妆品 |
	  | **ISO 8级** | - | - | 一般工业 |
	  | **ISO 9级** | - | - | 普通室内空气 |
	  
	  **关键认知**：
		- 等级每提升一级，洁净度提升**10倍**（粒子数减少到1/10）
		- 主流晶圆厂：ISO 4~ISO 5级（千级/万级）
		- 先进制程光刻区：ISO 1级（十级）
		- ISO 1级是什么概念？**一立方米空气中，0.1微米以上的灰尘不超过12颗**，比普通房间干净几百万倍
	- #### 2. 中国标准
		- GB 50073-2013《洁净厂房设计规范》
		- 基本等同采用ISO 14644-1
		- 保留了"百级、千级、万级、十万级"的传统叫法
- ### 三、半导体洁净室的技术要求
  
  洁净室不是"一个大房间"那么简单，是一个复杂的系统工程。
	- #### 1. 空气净化系统（核心）
		- **FFU（风机过滤单元）**：
			- 洁净室的"肺"，负责过滤空气
			- 由高效过滤器（HEPA/ULPA）+ 风机组成
			- 吊顶上密密麻麻铺满FFU
		- **过滤器等级**：
			- HEPA（高效过滤器）：过滤0.3μm粒子效率99.97%
			- ULPA（超高效过滤器）：过滤0.12μm粒子效率99.999%
			- 先进制程必须用ULPA级过滤器
		- **满铺率**：
			- ISO 5级（百级）：约50%满铺
			- ISO 3级（百级）：约80-90%满铺
			- ISO 1级（十级）：**100%满铺**（天花板全是FFU）
		- **换气次数**：
			- 万级：20-30次/小时
			- 千级：50-60次/小时
			- 百级：200-300次/小时
			- 十级：400-600次/小时
			- 意思是：整个房间的空气，每小时要全部过滤更新几百次
	- #### 2. 温湿度控制
		- **精度要求**：
		- 温度：±0.1℃ ~ ±0.5℃
		- 湿度：±1% ~ ±3% RH
		  
		  **为什么要这么精确**：
			- 温度变化会导致晶圆、设备热胀冷缩，影响光刻精度
			- 湿度太低容易产生静电，太高容易发霉、金属氧化
	- #### 3. 压差控制
		- 洁净度高的房间，压力要比洁净度低的房间高
		- 压差：10-15 Pa（相邻房间之间）
		- 目的：防止脏空气从低洁净区流向高洁净区
		- 空气永远是从干净的地方流向"脏"的地方
	- #### 4. 防静电（ESD）控制
		- 静电会吸附灰尘，还会击穿芯片
		- 地面、墙面、工作台都要用导静电材料
		- 接地电阻控制在10^5 ~ 10^9 Ω之间
		- 人员要穿防静电服、防静电鞋、戴防静电手环
	- #### 5. 防微振
		- 芯片制造设备对振动极其敏感
		- 光刻机的精度是纳米级的，一点点振动就会导致曝光偏差
		- 要求：振动速度VC-E级（0.125 μm/s）或更高
		- 措施：独立基础、隔振沟、空气弹簧、设备隔振器
	- #### 6. 气流组织
		- 高级别洁净室用**单向流（层流）**：空气从上往下匀速流动，像"空气活塞"一样把灰尘压下去
		- 低级别用**非单向流（乱流）**：靠稀释作用降低灰尘浓度
		- 气流速度：0.3-0.5 m/s（单向流）
	- #### 7. 其他控制
		- 噪音：≤65 dB
		- 照度：300-500 lux
		- 微生物控制（生物医药洁净室更严格）
		- AMC（气态分子污染物）控制：先进制程需要，空气中的化学分子也会影响芯片
- ### 四、洁净室的造价
  
  **洁净室是"黄金铺地"的工程**，等级越高，造价越贵。
  
  | 洁净等级 | 单位造价（含机电） | 说明 |
  |---------|-------------------|------|
  | **万级（ISO 6）** | 0.5-0.8万元/㎡ | 普通电子、医药 |
  | **千级（ISO 5）** | 1.0-1.5万元/㎡ | 成熟制程晶圆厂 |
  | **百级（ISO 4/5）** | 1.8-2.5万元/㎡ | 主流晶圆厂 |
  | **十级（ISO 1/2）** | 3.0-3.5万元/㎡ | 先进制程光刻区 |
  
  **造价差异的核心原因**：
  1. **FFU数量**：等级越高，FFU满铺率越高，设备成本翻倍
  2. **过滤器等级**：ULPA比HEPA贵很多
  3. **空调系统**：换气次数越多，空调机组越大，能耗越高
  4. **建筑结构**：防微振、气密要求高，建筑成本高
  5. **控制系统**：温湿度、压差、粒子监控等自动化系统
  
  **一个12英寸晶圆厂的洁净室投资**：
	- 洁净室面积：约3-5万㎡
	- 洁净室工程投资：约10-20亿美元
	- 占晶圆厂总投资的**15%-20%**
	- 是半导体产能扩张的"先行指标"
	  
	  **运营成本也很高**：
		- 电费：洁净室的空调+FFU是耗电大户，占晶圆厂用电的30-40%
		- 过滤器更换：ULPA过滤器1-2年就要换一次
		- 维护人员：专业的洁净室运维团队
- ### 五、洁净室产业链
	- #### 上游：设备和材料
	  
	  | 类别 | 代表产品 | 主要厂商 |
	  |------|---------|---------|
	  | **空气过滤设备** | FFU、HEPA/ULPA过滤器 | 外资：AAF、Camfil、康斐尔 <br> 国产： 美埃科技 、中建南方、富泰科技 |
	  | **空调机组** | 洁净空调、MAU、AHU | 外资：约克、开利、特灵<br>国产：美的、格力、 申菱环境 |
	  | **洁净室板材** | 彩钢板、洁净门、洁净窗 | 国产为主，竞争充分 |
	  | **高纯流体** | 纯水系统、特气系统、化学品供应 | 外资居多，国产替代中 |
	  | **防静电材料** | 防静电地板、墙面、服装 | 国产为主 |
	  | **AMC治理** | 化学过滤器、VOC处理 | 盛剑科技等 |
	  | **监控系统** | 粒子计数器、温湿度传感器、压差传感器 | 外资：TSI、MetOne<br>国产：苏信、赛多利斯 |
	- #### 中游：工程总包（EPC）⭐
	  
	  这是产业链的核心环节，也是业绩弹性最大的环节。
	  
	  **业务模式**：
		- 设计（Design）+ 采购（Procurement）+ 施工（Construction）
		- 从厂房设计到设备安装调试，交钥匙工程
		  
		  **主要玩家**：
			- 十一科技（太极实业）：国内半导体洁净室龙头，市占60-70%
			- 亚翔集成：台资，高端先进制程第一梯队
			- 中电二公司（深桑达A）：央企，实力强
			- 圣晖集成：台资，近年发展快
			- 其他：柏诚股份、华康世纪、中国电子系统等
	- #### 下游：应用领域
	  
	  | 领域 | 占比 | 特点 |
	  |------|------|------|
	  | **电子/半导体** | ~42% | 等级最高，投资最大，增速最快 |
	  | **生物医药** | ~20% | GMP标准，关注微生物控制 |
	  | **食品/化妆品** | ~15% | 等级相对低 |
	  | **航空航天** | ~10% | 精密制造 |
	  | **其他** | ~13% | 科研、医院手术室等 |
- ### 六、市场规模与竞争格局
	- #### 1. 市场规模
		- **2026年中国洁净行业整体规模**：约3800亿元，年复合增长率11.8%
		- **洁净室EPC市场**：约1642亿元，占整体43.2%
		- **半导体洁净室**：约400-600亿元，是增速最快的细分领域
		  
		  **驱动因素**：
		  1. **AI算力爆发**：带动晶圆厂扩产，先进制程占比提升
		  2. **存储芯片扩产**：长江存储、长鑫存储持续扩产
		  3. **国产替代**：半导体设备、材料国产化，带动配套洁净室需求
		  4. **生物医药**：创新药、CDMO扩产
		  5. **标准升级**：ISO 14644-1:2024新标准实施，0.1μm成为关注粒径，既有车间需要升级
- #### 2. 半导体洁净室竞争格局
	- **第一梯队（高端先进制程）**：
		- **亚翔集成**：台资，技术最强，ISO 1级能力，海外项目多
		- **十一科技（太极实业）**：国资，规模最大，国内市占最高
	- **第二梯队（中高端）**：
		- **中电二（深桑达A）**：央企，综合实力强
		- **圣晖集成**：台资，增速快
	- **第三梯队（中低端）**：
		- 柏诚股份、华康世纪等众多中小工程公司
	- **市场份额（不同统计口径有差异）**：
		- 十一科技：22-26%（或60-70%，看统计范围）
		- 中电二：17-19%
		- 亚翔集成：14-15%
		- 其他：40%+

- ### 八、为什么洁净室是AI时代的"隐形卡脖子"环节
	- 1. **扩产瓶颈**：
		- 全球AI算力爆发，晶圆厂疯狂扩产
		- 但高端洁净室EPC产能有限，能做ISO 1级的就那么几家
		- 洁净室建设周期1-2年，是产能扩张的瓶颈之一
	- 2. **标准升级**：
		- ISO 14644-1:2024新标准，关注粒径从0.5μm降到0.1μm
		- 很多旧车间在新标准下可能不达标
		- 需要升级改造，带来增量需求
	- 3. **先进制程占比提升**：
		- 7nm、5nm、3nm制程占比越来越高
		- 需要更高等级的洁净室（ISO 1级）
		- 单位造价翻倍，市场空间比单纯扩产更大
	- 4. **国产替代**：
		- 之前高端洁净室设备、材料主要靠进口
		- 现在国产替代加速，国内厂商份额提升
		- 从EPC到设备材料，全产业链都有机会
- ### 九、一句话总结
  
  **洁净室是芯片制造的"无菌手术室"，按每立方米灰尘数量分为ISO 1-9级，先进制程光刻区要达到ISO 1级（每立方米0.1μm灰尘不超过12颗），单平米造价高达3万元。产业链分为上游设备材料、中游工程总包（EPC）、下游应用三大环节，其中中游EPC是业绩弹性最大的环节，国内龙头是太极实业（十一科技）、亚翔集成等。AI算力爆发带动晶圆厂扩产，叠加标准升级和先进制程占比提升，洁净室行业正处于高景气周期。**
